# Interfaz-Grafica-MySQL
Para esta actividad generamos una conexion de un interfaz grafico a una base de datos creada en MySQL.

Para esto creamos una interfaz donde se obtuvieran los datos requeridos para hacer la conexion como lo serian Usuario, Contraseña, Host, Base de Datos y Puerto.
Esto para entrar dentro de la base de datos. Estos los acfomodamos de la siguiente manera en el interfaz y le agregamos un diseño. Esto seria unicamente cosas visuales.
